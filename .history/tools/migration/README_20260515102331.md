# Migration ciblée PS 1.6.1.24 -> PS 8.0.1

Ce dossier contient un kit de migration **sélective** pour:

- catégories
- produits
- prix (dont `specific_price`)
- taxes
- URLs SEO / friendly URLs

## Hypothèses prises

- Base cible PS8: déjà installée et fonctionnelle.
- Boutique cible vide en produits (confirmé dans l’état actuel).
- Langue unique en cible: Français `id_lang=1`.
- Langue source à migrer: Français `id_lang=2`.
- Dump source disponible: `old_db_prestashop 1_6_1_24.sql`.
- Le dump est d’abord importé dans une base intermédiaire nommée `legacy_ps16`.

## Fichiers

- `sql/01_migration_ps16_to_ps8.sql` : migration principale avec remapping des IDs de taxes.
- `sql/02_validation_queries.sql` : requêtes de contrôle post-import.
- `analyze_schema.php`, `compare_legacy_to_ps8.php`, `profile_target_ps8.php` : scripts d’analyse (audit technique).

## Stratégie recommandée (sécurisée)

1. **Sauvegarde complète** de la base cible PS8.
2. Créer la base intermédiaire `legacy_ps16` et y importer le dump 1.6.
3. Exécuter `sql/01_migration_ps16_to_ps8.sql` sur la base cible PS8.
4. Exécuter `sql/02_validation_queries.sql`.
5. Vider le cache PrestaShop et reconstruire l’index de recherche.
6. Vérifier BO + FO (catégories, fiches produit, prix TTC/HT, URLs).

## Points techniques importants

- Les **IDs de taxes** et **groupes de taxes** sont remappés automatiquement pour éviter les collisions avec PS8.
- Les **IDs catégorie/produit** sont conservés (safe ici car cible vide en produits et catégories standards limitées).
- Les valeurs `redirect_type` PS1.6 (`301`,`302`,`404`,`''`) sont converties au format PS8.
- Les dates invalides `0000-00-00` sont normalisées pour respecter les contraintes PS8.
- Les textes FR sont filtrés avec la règle `id_lang 2 -> 1`.

## Optionnel

Le script contient un bloc commenté pour reprendre `ps_shop_url` depuis la source.
Ne l’activer que si vous voulez remplacer l’URL locale actuelle.

## Rollback

En cas de problème:

- restaurer le backup SQL de la base cible;
- corriger la requête concernée;
- relancer la migration.
